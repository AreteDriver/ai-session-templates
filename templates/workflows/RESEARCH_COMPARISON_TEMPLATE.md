# Research / Comparison Template

Use this when comparing libraries, frameworks, APIs, or technical approaches before making
a decision. Forces structured evaluation criteria and surfaces hidden risks.

```md
# Research Session

## Decision To Make
[What are we choosing between?]

## Options
- [option A]
- [option B]
- [option C]

## Evaluation Criteria
- implementation speed
- reliability
- maintenance cost
- performance
- learning curve
- ecosystem maturity
- lock-in risk
- production suitability

## Constraints
[team size, timeline, infra, language, cost, deployment, etc.]

## Required Output
- comparison table or structured breakdown
- recommended option
- tradeoffs
- implementation advice
- hidden risks
```
